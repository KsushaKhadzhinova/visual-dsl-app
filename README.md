DiagramCode — Visual DSL DiagrammerПрофессиональная веб-среда для визуального моделирования диаграмм с использованием предметно-ориентированных языков (DSL). Проект разработан в рамках цикла лабораторных работ.Исполнитель: Ксения ХаджиноваВариант: 13Технологический стекHTML5: Семантическая разметка, WAI-ARIA, Schema.org.CSS3: Методология БЭМ, Flexbox, CSS Grid, Custom Properties.JavaScript: ES6+ Modules, Fetch API, LocalStorage, BOM API.Библиотеки: Lucide Icons (иконки), Mermaid.js / PlantUML (движки рендеринга).Этапы разработкиЛабораторная работа №6: Асинхронность и внешние APIРеализована модульная архитектура приложения на базе ES6 Modules.Интегрировано внешнее API для динамической загрузки шаблонов диаграмм.Внедрен мониторинг статуса сетевого подключения (Online/Offline) через BOM API.Обеспечен асинхронный рендеринг диаграмм с обработкой ошибок.Лабораторная работа №5: JavaScript и интерактивностьРеализована валидация форм с использованием регулярных выражений.Настроено сохранение состояния редактора и настроек пользователя в LocalStorage.Внедрена система уведомлений (Toasts) и управление модальными окнами.Применен паттерн делегирования событий.Лабораторная работа №4: Адаптивная версткаИнтерфейс полностью переведен на методологию БЭМ.Реализована адаптивная сетка (Mobile First) с использованием CSS Grid и Flexbox.Создано выпадающее мобильное меню (Drawer).Лабораторная работа №3: Семантика и доступностьВыполнена логическая разметка документа.Добавлены ARIA-атрибуты для поддержки скринридеров.Реализована навигация с клавиатуры (Skip-link).Лабораторная работа №2: ИнфраструктураНастроена среда Node.js и менеджер пакетов npm.Настроены скрипты автоматизации и игнорирование Git.Структура проекта/

│   ├── styles.css         # Глобальные стили

│   ├── main.css           # Редактор и холст

├── index.html             # Разметка
